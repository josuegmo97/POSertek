    Index de Categoria

    HTTP Request

    [POST] api/categoria/index

    En esta URL puedes verificar todas las categorias existentes

    Creacion de Categoria

    HTTP Request

    [POST] api/categoria/create

    Parametros de Entrada

    Parametro Tipo Status Descripcion
    categoria string required Nombre de la Categoria.
    cod_categoria string required Codigo de Categoria.
    descripcion string optional Descripcion de la Categoria.
    sub_categoria boolean required Indica si esta es una Sub categoria.
    cat_principal integer optional Indica cual es la categoria principal.

    Modificacion de Categoria

    HTTP Request

    [POST] api/categoria/update

    Parametros de Entrada

    Parametro Tipo Status Descripcion
    id_categoria string required ID de la Categoria que desea editar.
    categoria string required Nombre de la Categoria.
    cod_categoria string required Codigo de Categoria.
    descripcion string optional Descripcion de la Categoria.
    sub_categoria boolean required Indica si esta es una Sub categoria.
    cat_principal integer optional Indica cual es la categoria principal.

    Index de Variaciones

    HTTP Request

    [POST] api/variaciones/index

    En esta URL puedes verificar todas las variaciones existentes

    Creacion de Variaciones

    Ejemplo de Request:

        
        {
            "descripcion": "Colores",
            "valores": ["Azul","Rojo","Verde"]
        }

    HTTP Request

    [POST] api/variaciones/create

    Parametros de Entrada

    Parametro Tipo Status Descripcion
    descripcion string required Descripcion de la Variacion.
    valores array required Array con todas las variaciones existentes.

    Edicion de Variaciones

    Ejemplo de Request:

        
        {
            "id": 1,
            "descripcion": "Color",
            "valores": [
                {
                    "id": 1,
                    "descripcion": "Azul"
                },
                {
                    "id": 2,
                    "descripcion": "Rojo"
                },
                {
                    "id": 3,
                    "descripcion": "Verde"
                }
            ]
        }

    HTTP Request

    [POST] api/variaciones/create

    Parametros de Entrada

    Parametro Tipo Status Descripcion
    id integer required ID de la Variacion.
    descripcion string required Descripcion de la Variacion.
    valores array required Array con todas las variaciones existentes.
    valores.id array required ID dentro del Array de Valores.
    valores.descripcion array required Descripcion dentro del Array de Valores.
